◆ cu_add_fd()

int cu_add_fd ( int  fd,
uint32_t  opts,
cu_reactor_t  reactor 
)

Adds a file descriptor to polling list. cu_add_fd needs before cu_read, cu_write operations. Afters usage, the file must be closed by cu_close.

Parameters
fd- file descriptor
opts- no zero opts

Flags:

  • CU_IN - polling input events
  • CU_OUT - polling output events

The flags can be unite by or | operator.

Parameters
reactor
Returns
on success, returns 0; on error, returns -1 and the error set appropriately.
